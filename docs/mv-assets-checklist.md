## MV 공용 리소스/필증 확인 체크리스트 (테스트용)

1. **가이드 노출**: 사용자 접수 상세에서 `표기 가이드 다운로드` 버튼이 항상 보이고, 클릭 시 서명 URL로 열리는지 확인한다.
2. **등급 이미지 매핑**: 관리자가 등급을 `12`로 설정 후 사용자 화면에서 `등급 이미지 다운로드`가 12 로고를 반환하는지 확인한다. (ALL/12/15/18 전부 시도)
3. **필증 업로드/다운로드**  
   - 관리자 상세에서 파일 업로드 → 성공 알림 후 메타데이터가 저장되는지.  
   - 사용자 상세에서 상태가 `RESULT_READY` 또는 `COMPLETED`일 때만 `심의 필증 다운로드`가 노출되고 정상 다운로드되는지.
4. **필증 미존재 시**: 필증이 없거나 상태가 준비 전이면 버튼이 숨김/비활성인지 확인한다.
5. **권한 차단**: 비관리자가 admin API를 호출하면 403, 소유자가 아닌 사용자가 다운로드 API를 호출하면 403/404가 반환되는지 확인한다.

> 공용 리소스 objectKey는 코드에 고정 매핑(ENV 우선, 기본값 fallback)으로 관리하며, submission_files 테이블과 연결하지 않는다.
